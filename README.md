# Feather Fury

Feather Fury is an Angry Birds-inspired browser game implemented with plain JavaScript and HTML canvas.

## Features

- Physics-based slingshot launching.
- Five bird archetypes with active abilities.
- Material-based destructible blocks.
- Pigs with collision-based damage.
- 300 campaign levels.

## Run locally

Because this project uses ES modules, serve it over HTTP:

```bash
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Controls

- Mouse/touch drag near slingshot to aim.
- Release to launch.
- `Space` to trigger active ability.
- `R` to restart level.
